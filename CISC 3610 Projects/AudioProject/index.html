Voice Recognition Project

Instructions

Press the button and say any of the following commands. Once you are done, you can stop the speak button by clicking on it. Alternatively, if the program detects a pause it will stop it for you.

Say "color _____" to change the color of the circle.For example "color blue" to make the circle blue.

Say "background _____" to change the background color of the canvas. For example "background yellow" to make the background yellow.

Say "size _____" to change the size of the circle. For example "size 50" to make the circle with a diameter of 50.

Say "Help", if you need audio instructions

Say "About", if you'd like to hear more about this program and the developer behind it.
