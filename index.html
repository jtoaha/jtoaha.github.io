Game Design

Spring 2020

Source Code: https://github.com/jtoaha/jtoaha.github.io

Solo Project
galactic diesel etcI created this game for my Game Design course. Luna's goal is to collect galactic dots to refuel her tank. Implemented a score meter that resembles a tank being filled based on number of galactic dots collected. Also, implements different features, such as drawing Besier and Spline curves, pressing specified keys allow red, green or blue tints to be applied to an image, etc. Luna was taken from a previous Processing project (Java) of mine and was converted and scaled to be compatible with p5.js and the goals of this game. Launched on openprocessing.org
Group Project
revamped DREIDEL!My team worked on creating a revamped version of the game, Dreidel. This game alternates between spinning the dreidel and playing a memory matching card game to distribute coins. In this project, I was team lead and brought together everyone's ideas and contributions. We were lucky to have a team member who designed a 3D model of the Dreidel. On my part, I also contributed where necessary. Largely responsible for implementing game logic, game play, and designing the cards using Adobe Illustrator. Created using p5.js and launched on the platform, openprocessing.org.
